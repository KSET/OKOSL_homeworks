Projekt je napisan koristeći Flask, jedan od najpopularnijih Python web frameworkova.
Za pristup bazi korišten je driver PyMongo.

Projekt se pokreće iz shella:
	1) prvo je uputno pokrenuti virtualno okruženje: source venv/bin/activate,
	2) potom se naredbom pip install -r requirements.txt dohvaćaju svi potrebni paketi,
	3) tada se pokretanjem naredbe "flask run" u korijenskom direktoriju (mongo_portal) pokreće program.

Pristup programu iz web preglednika: localhost:5000/index
